Welcome to Dhara , a geo science gateway.


Dhara provides access to geo science services by extracting geo data from various geo data sources and analyzing them. It also provides an environment where it is possible for the scientists to create, execute, manage and test geo science workflows.


Scientists can use geo science services catered through Dhara to access and analyze geo spatial data. Dhara creates an environment to run geo science experiments such as spatial prediction of landslide hazards, with its workflow management capabilities.

Introduction

Integration of capabilities to support geo services into a scientific workflow management system plays a vital role in geo science research. Currently Apache Airavata has been used by scientific research groups for ease of use and exposes the capabilities of their existing scientific software, services and processes to other researchers. Dhara is a geo science gateway to enable the construction of geo science workflows that leverage standardized, publicly available geo services and geo data sources. We will follow Web Processing Service (WPS) Interface Standard which provides rules for standardizing inputs and outputs for geospatial processing services.


What is Apache Airavata ?

Apache Airavata is a software framework which provides the environment to compose, manage, execute and monitor scientific applications and workflows on a range of computing resources such as local clusters, grids and clouds. Apache Airavata features include; a desktop tool and browser based web interface to manage applications and workflows, server-side tools for registering and managing scientific applications, a graphical user interface to manage and reuse scientific workflows, interfacing and interoperability with various third party tools.

What is OGC ?

The Open Geospatial Consortium (OGC) is an international industry consortium of around 500 companies, government agencies and universities. They work on developing publicly available interface standards that support interoperable solutions that 'geo-enable' the Web, wireless and location based services and mainstream IT. OGC standards are available as services on GeoServer and MapServer and include the following:

  • WPS (Web Processing Service)
  • WCS (Web Coverage Service)
  • WFS (Web Feature Service)
  • WMS (Web Map Service)
